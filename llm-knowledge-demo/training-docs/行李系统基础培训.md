# 行李系统基础培训

行李输送系统用于把旅客托运行李从值机柜台输送到分拣区域，再送往对应航班的装载位置。系统通常包括值机注入线、主输送线、分拣机、滑槽、安检设备、到港转盘和人工处理点。

日常巡检应关注输送带运行声音、皮带跑偏、急停按钮状态、光电传感器遮挡、分拣口堵包和控制柜报警。发现异常时，先确认现场是否存在人员安全风险，再按流程通知运行控制岗位。

如果输送线局部堵包，应先暂停相关区域设备，清理卡阻行李，检查标签、绑带或异物是否卷入滚筒。恢复前需要确认防护罩、急停和人员位置均处于安全状态。

系统异常记录应包含时间、位置、设备编号、现象、处理人和恢复时间。记录越完整，后续复盘和设备维护越容易定位原因。
