# Baggage Point Finder

本项目是一个本地运行的行李输送机点位标注工具。

## 当前功能

- 按图纸切换查看楼层或总览图。
- 在画布上创建、移动、删除点位标注。
- 点位可命名、备注，并支持实时搜索定位。
- 右侧点位总览支持分组、新建分组、重命名、折叠/展开、删除分组。
- 支持勾选多个点位后批量拖动或点击分组归组。
- 支持自动命名、JSON 导入导出、本地存储和自动备份。
- 点位图纸从 `pdf/` 自动生成，新增或删除 PDF 后会更新图纸列表。
- 培训资料从 `training-docs/` 自动生成目录，支持 DOCX、PDF、Excel、CSV、PPTX 的正文或表格内容解析。

## 代码结构

- `index.html`：页面结构和控件入口。
- `styles.css`：整体布局、画布、侧栏、分组列表和点位样式。
- `app.js`：应用状态、点位绘制、搜索、分组、导入导出、自动命名和事件绑定。
- `app.js` 同时包含培训资料库的目录、文档上传、文件解析和阅读区逻辑。
- `assets/floors/`：图纸图片和清单。
- `pdf/`：本地图纸 PDF 来源目录。
- `training-docs/`：本地培训资料来源目录，子文件夹会映射为资料目录组。
- `scripts/`：辅助脚本。

## 本地使用

```bash
npm start
```

启动后打开 `http://localhost:8082/`。把图纸 PDF 放进 `pdf/`，把培训资料放进 `training-docs/`；本地服务会监听变化并更新页面读取的清单。
